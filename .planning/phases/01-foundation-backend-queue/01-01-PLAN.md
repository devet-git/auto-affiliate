# Plan 01-01: Backend Foundation & Hardcoded Auth

## Objective
Khởi tạo FastAPI backend server với cấu trúc Domain-Driven, mô hình bảo mật JWT dùng tài khoản Admin cứng từ biến môi trường. Thỏa mãn requirement CORE-01 và CORE-03.

## Tasks

### 1. Initialize Requirements & Settings <automated>
- `<read_first>`: `.planning/phases/01-foundation-backend-queue/01-CONTEXT.md`
- `<action>`: 
  - Ghi vào `server/requirements.txt`: `fastapi`, `uvicorn`, `pydantic-settings`, `python-jose[cryptography]`, `passlib[bcrypt]`, `python-multipart`.
  - Ghi vào `.env.example`: `ADMIN_USERNAME=admin`, `ADMIN_PASSWORD_HASH=$2y$10$wN2aO8WdD35v2pC5W.RlxO1S8S/uPx37/G7qWdF.rK9y6G4v9b2p.`, `SECRET_KEY=yoursecret`.
  - Viết `server/app/core/config.py` dùng `pydantic_settings.BaseSettings`, định nghĩa `ADMIN_USERNAME`, `ADMIN_PASSWORD_HASH` và `SECRET_KEY`. Cấu hình `model_config = SettingsConfigDict(env_file=".env")`.
- `<acceptance_criteria>`: `server/app/core/config.py` export class Settings chứa 3 biến trên.

### 2. Implement Security Core <automated>
- `<read_first>`: `server/app/core/config.py`
- `<action>`: 
  - Viết `server/app/core/security.py`. Khởi tạo `pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")`.
  - Viết hàm `verify_password(plain_password: str, hashed_password: str) -> bool`.
  - Viết hàm `create_access_token(data: dict, expires_delta: timedelta) -> str` dùng thư viện `jose.jwt`.
- `<acceptance_criteria>`: `verify_password` và `create_access_token` tồn tại trong file `security.py`.

### 3. Build Admin Router & Login Flow <automated>
- `<read_first>`: `server/app/core/security.py`
- `<action>`: 
  - Tạo Schema Token: `server/app/domains/admin/schemas.py` chứa class `Token(BaseModel): access_token: str, token_type: str`.
  - Thiết lập Login Route: `server/app/domains/admin/router.py`. Tạo router `/login`. Trích thông tin từ `Depends(OAuth2PasswordRequestForm)`. So sánh `form_data.username` với `settings.ADMIN_USERNAME` và if `verify_password(form_data.password, settings.ADMIN_PASSWORD_HASH)` fail thì raise HTTP 401. Nếu đúng yield token.
- `<acceptance_criteria>`: `router.py exposes router object with POST /login`.

### 4. Setup Authentication Dependency <automated>
- `<read_first>`: `server/app/domains/admin/router.py`
- `<action>`: 
  - Tạo dependency `get_current_admin` ở `server/app/domains/admin/dependencies.py` dùng `OAuth2PasswordBearer(tokenUrl="api/v1/auth/login")`.
  - Viết logic decode token. Nếu token decode lỗi, raise 401. Trả về payload json `{ "sub": "admin" }` nếu thành công.
- `<acceptance_criteria>`: `dependencies.py` có hàm `get_current_admin`.

### 5. Finalize App Entrypoint <automated>
- `<read_first>`: `server/app/domains/admin/router.py`
- `<action>`: 
  - Ghi vào `server/main.py`: `from fastapi import FastAPI`. Include admin router với prefix `/api/v1/auth`.
  - Thêm một mock health check GET `/api/v1/health` trả về `{"status": "ok"}` để test.
- `<acceptance_criteria>`: `main.py imports app as FastAPI instance`
