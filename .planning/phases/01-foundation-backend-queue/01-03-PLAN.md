# Plan 01-03: SQLModel Database Integration

## Objective
Khởi tạo kết nối SQLModel tới PostgreSQL, thiết kế bảng `Campaign` căn bản (chưa cần dữ liệu thô). Thỏa mãn requirement CORE-04 (Database Schema).

## Tasks

### 1. Add SQLModel Dependencies <automated>
- `<read_first>`: `server/requirements.txt`
- `<action>`: 
  - Ghi vào `server/requirements.txt`: `sqlmodel`, `psycopg2-binary`, `alembic` (chuẩn bị).
  - Cập nhật `server/app/core/config.py` thêm `DATABASE_URL: str = "postgresql://postgres:postgres@localhost:5432/auto_affiliate"` vào `Settings`. Khai báo vào `.env.example`.
- `<acceptance_criteria>`: `server/requirements.txt contains sqlmodel`

### 2. Configure Database Session <automated>
- `<read_first>`: `server/app/core/config.py`
- `<action>`: 
  - Tạo `server/app/core/database.py`. Khởi tạo `engine = create_engine(settings.DATABASE_URL, echo=True)`.
  - Khai báo dependency function `get_session() -> Generator[Session, None, None]` dùng `sqlmodel.Session(engine)`.
  - Viết method `def create_db_and_tables(): SQLModel.metadata.create_all(engine)`.
- `<acceptance_criteria>`: `database.py configures create_engine using DATABASE_URL`

### 3. Create Campaign Model <automated>
- `<read_first>`: `server/app/core/database.py`
- `<action>`: 
  - Tạo `server/app/domains/campaign/models.py`. 
  - Kế thừa `SQLModel`, định nghĩa class `Campaign(SQLModel, table=True)` có ID UUID4 `Field(default_factory=uuid.uuid4, primary_key=True)`, `name: str`, `status: str = "draft"`, `created_at: datetime = Field(default_factory=datetime.utcnow)`.
- `<acceptance_criteria>`: `models.py imports SQLModel`

### 4. Create Tables on Startup <automated>
- `<read_first>`: `server/main.py`, `server/app/core/database.py`, `server/app/domains/campaign/models.py`
- `<action>`: 
  - Cập nhật `server/main.py`. Import `create_db_and_tables` và `Campaign` model (để metadata đăng ký bảng).
  - Dùng FastAPI lifespan events `@asynccontextmanager def lifespan(app: FastAPI): create_db_and_tables() yield`.
- `<acceptance_criteria>`: `main.py imports lifespan`

## Verification
- Khởi động service Postgresql (bằng Docker/Local).
- Chạy terminal `uvicorn server.main:app`. Log in ra: `CREATE TABLE campaign ...` thì kết nối SQLModel đã thành công hoàn chỉnh.
