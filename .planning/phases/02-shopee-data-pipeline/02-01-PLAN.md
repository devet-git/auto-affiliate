<task>
<description>Viết Core Playwright Scraper Engine</description>
<read_first>
- server/app/domains/shopee_crawler/models.py
</read_first>
<action>
Tạo file `server/app/domains/shopee_crawler/scraper.py`.
Import `sync_playwright` từ thư viện `playwright.sync_api`.
Xây dựng hàm `scrape_keyword(keyword: str, max_items: int = 50) -> list[dict]`:
- Start chromium HEADLESS=True (hoặc mode debugging nếu cần). Mở page tới `https://shopee.vn/search?keyword={keyword}`.
- Gọi loop javascript để scroll down nhiều lần đến đáy trang nhằm trigger lazy-load của ảnh và listing grid.
- Parse DOM elements list sản phẩm (dựa trên locator div class của Shopee): Trích xuất thẻ `img` src, text tựa đề, số tiền và URL chi tiết thẻ `a` href.
- Return danh sách Dict chứa các biến meta kể trên.
</action>
<acceptance_criteria>
`server/app/domains/shopee_crawler/scraper.py` defines `def scrape_keyword` function using `sync_playwright` internally
</acceptance_criteria>
</task>
