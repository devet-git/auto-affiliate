# Plan 02-02: Module Affiliate Link Generator

<objective>
Xây dựng module "Tà đạo" tận dụng Playwright để tự động đăng nhập (bằng cookie cung cấp trước) và trích xuất link Affiliate thẳng trên giao diện Shopee CMS thay vì dùng API chính thức. Điều này biến các products `PENDING` thành `CONVERTED` với affiliate URL xịn sò.
</objective>

<tasks>

<task>
<description>Phát triển chức năng Playwright Bulk Link Convert</description>
<read_first>
- server/app/domains/shopee_crawler/scraper.py
</read_first>
<action>
Tạo file `server/app/domains/shopee_crawler/affiliate.py`.
Viết function `convert_affiliate_links(urls: list[str], state_file: str) -> dict[str, str]`:
1) Chạy `sync_playwright` khởi tạo Chromium context truyền argument `storage_state=state_file`.
2) Truy cập `https://affiliate.shopee.vn/custom_link`.
3) Inject list urls này vào TextBox Custom Link (Tạo hàng loạt).
4) Bấm nút Submit (Generate).
5) Lấy list output Tracking Link trả về, map giữa Link Cũ -> Link Mới (vd: { "shopee.vn/ao": "shope.ee/xyz" }).
</action>
<acceptance_criteria>
`server/app/domains/shopee_crawler/affiliate.py` contains `def convert_affiliate_links` with Playwright `storage_state` usage
</acceptance_criteria>
</task>

<task>

<action>
